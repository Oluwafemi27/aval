# @pixel-point/aval-format

Strict parser, validator, writer, AVC inspector, PNG decoder, and graph adapter
for compiled AVAL wire format `0.1`. Hostile byte arithmetic,
layout, media profile, and digest gates remain checked without imposing a
small-asset product ceiling.

See the repository [format `0.1` reference](../../docs/format/0.1.md) and
[authoring guide](../../docs/compiler/authoring-video-and-states.md).
