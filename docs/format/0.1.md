# Compiled format 0.1

An `.avl` asset has a fixed header, canonical front-index manifest and sample
table, aligned payload layout, and independently decodable motion units. The
file ends after its final access-unit payload and contains no poster/static
image plane. All integer arithmetic, UTF-8, canonical JSON, offsets, overlaps,
geometry, AVC, and digest fields are validated before media use.

The graph supports authored states, body units, transitions, ports, portals,
finish/cut/locked routes, exact inverse pairs, bindings, and readiness.
Half-open frame ranges and rational frame rate define behavior; there is
no runtime seek instruction or embedded script.

Package `1.0.0` continues to read wire version `0.1`. Package semver and wire
version are independent compatibility spaces.

Wire 0.1 has no product-policy ceiling at the former prototype canvas, frame,
reversible-unit, or compiled-file defaults. It retains exact integer field
ranges, checked offset/length arithmetic, codec-profile conformance, and strict
allocation/device failure boundaries. Readers may apply an explicitly supplied
lower host policy, but must not silently alter authored media.
